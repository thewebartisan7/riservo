# Archive Guidance

This directory contains historical documentation and archived implementation plans.

- Do not read files in this subtree by default.
- Read archive files only if the user explicitly asks for historical context, or if the current task cannot be completed without checking prior archived material.
- Prefer active docs first: `docs/README.md`, `docs/HANDOFF.md`, `docs/ROADMAP.md`, `docs/SPEC.md`, `docs/DECISIONS.md`, and the topical files linked from there.
- If you do read an archive file, keep that reading narrow and task-specific.
