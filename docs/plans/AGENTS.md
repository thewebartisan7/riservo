# Plans Guidance

This directory contains active implementation plans and planning artifacts.

- Do not read multiple plan files by default.
- Read only the specific plan file explicitly requested by the user or clearly required by the current task.
- If no exact plan file is referenced, prefer the most directly relevant current plan instead of scanning the whole directory.
- For general project context, prefer active docs first: `docs/README.md`, `docs/HANDOFF.md`, `docs/ROADMAP.md`, `docs/SPEC.md`, and `docs/DECISIONS.md`.
